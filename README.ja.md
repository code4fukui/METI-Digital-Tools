# METI Digital Tools


このリポジトリには、日本の経済産業省（METI）デジタルトランスフォーメーション室が実施した、デジタルツールの導入および調査プロジェクトに関する報告書が収められています。

## 特徴
- 公共部門におけるデジタルツールの導入および調査プロジェクトの概要
- ITインフラをセキュアで使いやすいクラウド環境へ移行するための知見

## 要件
特定の要件はありません。

## 利用方法
本報告書は参考目的でのみ提供されています。過年度のプロジェクトの報告書であるため、プルリクエストはお受けできません。ただし、コメントやフィードバックは歓迎いたします。
